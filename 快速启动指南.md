# 🚀 Mozart AI ERP 快速启动指南

## 📋 前置要求

- Python 3.11+
- Node.js 18+
- PostgreSQL 14+（或使用SQLite）
- Redis（可选）

---

## ⚡ 快速启动（3步）

### 1️⃣ 配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，至少配置以下内容：
# - DEEPSEEK_API_KEY=your-deepseek-api-key-here
# - DATABASE_URL=sqlite:///./mozart.db  # 使用SQLite快速开始
# - SECRET_KEY=your-secret-key-min-32-chars
# - JWT_SECRET_KEY=your-jwt-secret-key-min-32-chars
```

### 2️⃣ 启动后端

```bash
# 进入后端目录
cd backend

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 初始化数据库
python init_db.py

# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 3️⃣ 启动前端

```bash
# 打开新终端，进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 🧪 测试示例

### 1. 用户注册和登录

访问 http://localhost:8000/docs，找到认证API：

**注册用户**：
```json
POST /api/v1/auth/register
{
  "username": "testuser",
  "email": "test@example.com",
  "password": "password123",
  "display_name": "测试用户"
}
```

**登录**：
```json
POST /api/v1/auth/login
{
  "username": "testuser",
  "password": "password123"
}
```

### 2. 提交贡献

使用获取的访问令牌：

```json
POST /api/v1/say
Headers: Authorization: Bearer <your_token>
{
  "actor_id": "<your_actor_id>",
  "message": "今天采购了20斤土豆35元一斤供应商张三"
}
```

### 3. 查询信息

```json
POST /api/v1/ask
Headers: Authorization: Bearer <your_token>
{
  "actor_id": "<your_actor_id>",
  "question": "本月采购了多少钱？"
}
```

### 4. 上传图片识别

```json
POST /api/v1/ocr/recognize/invoice
Headers: Authorization: Bearer <your_token>
Body: form-data
  - file: <图片文件>
```

### 5. 查看通知

```json
GET /api/v1/notifications/
Headers: Authorization: Bearer <your_token>
```

### 6. 查看排行榜

访问 `GET /api/v1/actors/leaderboard`

### 7. 运行测试

```bash
cd backend
pytest tests/
```

## 🔧 常见问题

### Q1: 数据库连接失败

**解决方案**: 使用SQLite快速开始
```bash
# 修改 .env 文件
DATABASE_URL=sqlite:///./mozart.db
```

### Q2: DeepSeek API调用失败

**解决方案**: 
1. 确认 `DEEPSEEK_API_KEY` 已正确配置
2. 检查API余额是否充足
3. 如果没有API key，系统会使用规则引擎处理（功能有限）

### Q3: 前端无法连接后端

**解决方案**: 
检查前端API配置 `frontend/src/api/index.ts`：
```typescript
const api = axios.create({
  baseURL: 'http://localhost:8000/api/v1',
  // ...
})
```

### Q4: 缺少依赖

**解决方案**:
```bash
# 后端
pip install -r requirements.txt

# 前端
npm install
```
