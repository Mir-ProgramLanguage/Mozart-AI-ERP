## 📊 项目概览

Mozart AI ERP 是一个**AI原生企业资源管理系统**，核心理念是：
- **人是核心资产** - 不是岗位工具，是能力集合
- **贡献即回报** - 打破固定薪酬，动态回报
- **多角色自由** - 一个人可以做任何贡献
- **价值量化** - AI评估每个贡献的价值

## 🔄 核心流程

### 1. 贡献提交流程
```
用户输入 → 意图识别 → 实体提取 → 贡献评估 → 任务生成 → 智能分配 → 存储 → 返回响应
```

**示例**:
```
输入: "今天采购了20斤土豆35元一斤供应商张三"

AI理解:
{
  "intent": "contribute",
  "sub_intent": "purchase",
  "entities": [
    {"type": "item", "value": "土豆", "raw_text": "土豆"},
    {"type": "quantity", "value": 20, "raw_text": "20斤"},
    {"type": "amount", "value": 700, "raw_text": "35元一斤"}
  ]
}

贡献评估:
{
  "type": "purchase_saving",
  "estimated_value": 210,  // 假设节省了30%
  "confidence": 0.85
}

任务生成:
[
  {
    "type": "verify",
    "description": "核实采购记录",
    "priority": 65,
    "assigned_to": "最佳匹配的Actor"
  }
]
```

### 2. 回报计算流程
```
贡献验证 → 计算回报 → 发放回报 → Actor余额更新
```

**示例**:
```
贡献价值: 1000元

回报计算:
- 现金奖励: 100元 (1000 × 10%)
- 利润分红: 400元 (1000 × 40%)
- 技能点: 1点 (1000 / 1000)

总回报: 500元现金 + 1技能点
```

### 3. 能力成长流程
```
贡献完成 → 能力成长 → 更新能力图谱 → 记录成长历史
```

**示例**:
```
当前能力: 采购 = 0.7
贡献价值: 1000元

成长计算:
delta = 0.01 × (1 + 1000/10000) × (1 - 0.7)
      = 0.01 × 1.1 × 0.3
      = 0.0033

新能力值: 0.7033
```

### 1. 环境准备

#### 后端
```bash

```
