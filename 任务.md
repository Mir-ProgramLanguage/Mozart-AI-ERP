# Mozart AI ERP 开发任务

## 项目愿景
> 人是核心资产，贡献即回报 - 打破岗位边界，释放组织活力

## 核心理念

**传统企业**：
- 员工 = 岗位角色
- 只能做岗位职责内的事
- 固定工资，贡献难以量化

**AI原生企业**：
- 人 = 核心资产（能力集合）
- 所有能力都可以发挥
- 每一滴付出都有回报

## 核心架构

```
Actor（能力个体）→ 贡献 → AI评估价值 → 回报
                ↓
              任务流 → 协作 → Story（事件流）
```

**五大支柱**：
1. **Actor** - 能力个体（多维度能力图谱）
2. **Contribution** - 贡献系统（所有事件都是贡献）
3. **Evaluation** - 价值评估（AI量化贡献价值）
4. **Reward** - 回报机制（贡献值转换为回报）
5. **Growth** - 能力成长（贡献促进能力提升）

## 核心价值主张

### 1. 每一滴付出都有回报
```
发现采购机会 → 贡献值 +90 → 回报 9-36元
完成销售 → 贡献值 = 销售额 × 利润率 × 50%
推荐人才 → 贡献值 2000-10000元
```

### 2. 打破岗位边界
```
开发可以：
- 做开发 → 贡献值 +200/天
- 发现市场机会 → 贡献值 +500
- 推荐人才 → 贡献值 +2000

HR可以：
- 招聘 → 贡献值 +5000
- 卖产品 → 贡献值 = 销售额 × 5%
```

### 3. 能力成长可视化
```
每次贡献 → 能力提升
市场洞察: 0.70 → 0.71
采购优化: 新增能力标签
```

### 4. 公平透明
```
所有贡献都有记录
所有回报都有依据
排行榜实时更新
```
